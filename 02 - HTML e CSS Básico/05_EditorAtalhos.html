Editor Atalhos

Quando usa o Ctrl+C sem nada selecionado ele Copia / Duplica a linha inteira.

Usando Alt ↑ ou Alt ↓ ele move a linha pra cima ou para baixo.

Usando Crtl D depois de ter copiado uma primeira palavra ele vai selecionar as outras iguais

Atalho para criar um comentario: Ctrl ; Se usar o comando na linha, a linha toda é comentada

Ctrl B tira o aba do lado esquerdo.

Ele ensina a criar Atalho tambem.
